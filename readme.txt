Project Name:
--------------
Student Management System (Web Application)

Project Description:
---------------------
The Student Management System is a simple web-based application developed using HTML, CSS, Bootstrap, JavaScript, and Python. The application allows users to manage student records by performing CRUD (Create, Read, Update, Delete) operations through a user-friendly interface.

The project is designed for beginners to understand frontend and backend integration using Python.

-----------------------------------------------------------
TECH STACK
-----------------------------------------------------------

Frontend:
---------
• HTML5
• CSS3
• Bootstrap 5
• JavaScript (Vanilla JS)

Backend:
--------
• Python

(No Database Required)
Student records can be stored in:
• JSON File
OR
• Text File

• Required fields cannot be blank
