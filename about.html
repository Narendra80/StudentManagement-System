Architecture & Learning Outcomes

About The Project

The Student Management System is a comprehensive full-stack web application developed to demonstrate seamless frontend and backend integration using HTML5, Bootstrap 5, Vanilla JavaScript, and Python.

Technology Stack

  • Frontend Architecture HTML5 semantic structure, Bootstrap 5 responsive grid & components, custom glassmorphic CSS tokens, and Vanilla JavaScript DOM manipulation & asynchronous API calls.
  • Python Backend & API Powered by Python 3 and Flask. Serves static files cleanly and provides RESTful endpoints (`GET`, `POST`, `PUT`, `DELETE`) with robust server-side regex validation.
  • JSON File Storage (No DB Setup Required) All records are persisted locally in `python/students.json` using Python's native file handling and JSON module, ensuring zero database installation friction.

Key Highlighted Features

Dark/Light Theme Persistent theme toggling with custom CSS variables and smooth transitions.
Regex Validation Strict client and server checking for email format, 10-digit mobile, and unique IDs.
Export & Print One-click JSON database export and browser-friendly stylesheet printing.
Live Autocomplete Debounced search querying across multiple fields with interactive suggestion chips.

Application Workflow

1. Launch Python Server 2. Open Dashboard 3. Perform CRUD Operations 4. Real-time JSON Persistence
